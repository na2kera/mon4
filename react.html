useState

useStateの説明

関数コンポーネントでstateを管理（stateの保持や更新）するための
Reactフック。

使用例

const [count, setCount] = useState(0);
setCount(count + 1);

コードの説明

useStateの引数で初期化
setCountが呼び出されるごとに変数を更新。
ある関数内でも呼び出せる。

useRef

useRefの説明

DOMを参照して要素にアクセスするためのhooks
レンダリングを起こさず値を保持する

使用例

const ref = useRef(null);
console.log(ref.current.value);

コードの説明

useRefの引数で値を初期化。
ref変数に値を格納。
ref.current.valueで入力された値を取り出す。

useReducer

useReducerの説明

状態管理のためのフックで、useStateと似たような機能。
useStateはuseReducerに内部実装されている。

使用例

const [state, dispatch] = useReducer(reducer, 0);
() => dispatch({ type: "increment" })

コードの説明

useReducerの第二引数に初期値を設定。
dispatchは、reducerを実行するための呼び出し関数。

useMemo

useMemoの説明

関数の結果を保持するためのフックで、
何回やっても結果が同じ場合の値などを保存(メモ化)する。

使用例

useMemo(() => {
//処理をかく
}, [count02]);

コードの説明

結果を保持したい処理をuseMemo()で囲う。
第二引数には値が更新されるかどうかの対象を指定。
